Let f(x) = \dfrac{format(PROBLEM, "large")}{x + b}. Note that f(x) is undefined at x=-b. However, in this case, we can find a nicer function g(x) defined at x=-b that agrees with f(x) near x=-b.

Find the value of \displaystyle\lim_{x\to -b} f(x).

a-b

First factor the numerator.

We want to write format(PROBLEM, "large") in the form (x+\color{blue}{a})(x+\color{red}{b})

This requires that

\color{blue}{a} \cdot \color{red}{b} = a*b

\color{blue}{a} + \color{red}{b} = a+b

The following values can be used:

parseFormat("#{a}=#{" + a + "}", [BLUE, BLUE])
parseFormat("#{b}=#{" + b + "}", [RED, RED])

Thus format(PROBLEM, "large") = (x+\color{blue}{a})(x+\color{red}{b})

We see that

f(x) = \dfrac{format(PROBLEM, "large")}{x + b}= \dfrac{(x+a)(x+b)}{x + b}

Whenever x\ne-b, we can cancel the factor of x+b, so we have

f(x) = x+a for all x\ne-b

Thus g(x) = x+a is a continuous function that agrees with f everywhere except x = -b, and so \displaystyle\lim_{x \to -b}f(x)=g(-b) = a-b

Suppose that f(X) = Y, and f'(X) = M.

Using this information you can determine one and only one value of the derivative of f^{-1}(x).

(f^{-1})'(Y) =1/M
  • (f^{-1})'(Y) =1/M
  • (f^{-1})'(Y) =M
  • (f^{-1})'(X) =M
  • (f^{-1})'(X) =1/M
  • (f^{-1})'(X) =Y
  • (f^{-1})'(X) =1/Y

The inverse function rule says that (f^{-1})'(x)= \dfrac{1}{f'(f^{-1}(x))}

The only value we have for f is f(X) = Y, so the only value for f^{-1} we have is f^{-1}(Y) = X

Therefore (f^{-1})'(Y)= \dfrac{1}{f'(f^{-1}(Y))}

So (f^{-1})'(Y)= \dfrac{1}{f'(X)}

So (f^{-1})'(Y)= \dfrac{toFraction(1/M)[0]}{toFraction(1/M)[1]}

Find \displaystyle \int Ax\sqrt[n]{x+B} dx

Let u = x+B

Then \dfrac{du}{dx} = 1, so du = dx. Also note that x = u-B

Rewriting the integral with this substitution we have A\displaystyle\int (u-B)\sqrt[n]{u}du

=A \displaystyle\int (u-B)u^{\frac{1}{n}}du

= A\displaystyle\int \left( u^{\frac{n+1}{n}}-Bu^{\frac{1}{n}} \right) du

= A\left(\dfrac{n}{2*n+1}u^{\frac{2*n+1}{n}}-\dfrac{B*n}{n+1}u^{\frac{n+1}{n}}\right)
